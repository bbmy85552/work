# 问题一：单次反射透射干涉数学模型理论推导

## 一、物理场景分析

### 系统组成
考虑三层介质系统：
- **介质1**：空气（折射率 n₁ ≈ 1.0）
- **介质2**：碳化硅外延层（折射率 n₂，厚度 d）
- **介质3**：碳化硅衬底（折射率 n₃）

### 光路分析
红外光以入射角 θᵢ 从空气入射，在外延层内产生两条主要反射光束：
1. **光束1**：在空气-外延层界面（界面1-2）的直接反射光
2. **光束2**：透过界面1-2，在外延层-衬底界面（界面2-3）反射后再次透射出外延层的光束

## 二、理论推导

### 2.1 斯涅尔定律（折射定律）

根据斯涅尔定律，入射角与折射角的关系：
```
n₁ sin(θᵢ) = n₂ sin(θₜ)
```

其中：
- θᵢ：入射角
- θₜ：外延层内的折射角

### 2.2 光程差计算

光束2相对于光束1多传播的物理路程为在外延层内往返一次：
```
ΔL = 2 × n₂ × d × cos(θₜ)
```

### 2.3 相位差计算

相位差由光程差决定：
```
δ = (2π/λ) × ΔL = (2π/λ) × 2n₂dcos(θₜ)
```

转换为波数表示（k = 2π/λ）：
```
δ = 2k × n₂dcos(θₜ)
```

### 2.4 菲涅尔反射系数

对于TE偏振光（垂直入射面偏振）：
```
r₁₂ = (n₁cos(θᵢ) - n₂cos(θₜ)) / (n₁cos(θᵢ) + n₂cos(θₜ))
r₂₃ = (n₂cos(θₜ) - n₃cos(θ₃)) / (n₂cos(θₜ) + n₃cos(θ₃))
```

对于TM偏振光（平行入射面偏振）：
```
r₁₂ = (n₂cos(θᵢ) - n₁cos(θₜ)) / (n₂cos(θᵢ) + n₁cos(θₜ))
r₂₃ = (n₃cos(θₜ) - n₂cos(θ₃)) / (n₃cos(θₜ) + n₂cos(θ₃))
```

### 2.5 双光束干涉反射率

考虑双光束干涉，总反射光振幅：
```
Eᵣ = E₁ + E₂ = r₁₂Eᵢ + t₁₂t₂₁r₂₃e^(-iδ)Eᵢ
```

利用斯托克斯倒易关系：t₁₂t₂₁ = 1 - r₁₂²

总反射率：
```
R = |Eᵣ/Eᵢ|² = R₁ + (1-R₁)²R₂ + 2(1-R₁)√(R₁R₂)cos(δ)
```

其中：
- R₁ = r₁₂²：界面1-2的反射率
- R₂ = r₂₃²：界面2-3的反射率

### 2.6 干涉极大值条件

当相位差满足以下条件时，反射率出现极大值：
```
cos(δ) = 1 ⇒ δ = 2mπ (m = 0, 1, 2, ...)
```

即：
```
2kₘ × n₂dcos(θₜ) = 2mπ
kₘ × n₂dcos(θₜ) = mπ
```

### 2.7 厚度公式推导

对于相邻两个干涉峰（级次 m 和 m+1）：
```
kₘ × n₂dcos(θₜ) = mπ
kₘ₊₁ × n₂dcos(θₜ) = (m+1)π
```

两式相减：
```
(kₘ₊₁ - kₘ) × n₂dcos(θₜ) = π
Δk × n₂dcos(θₜ) = π
```

因此，外延层厚度：
```
d = π / (n₂cos(θₜ) × Δk)
```

其中 Δk 是相邻干涉峰的波数差。

### 2.8 波数与波长的关系

由于波数 k = 2π/λ，波数差与波长差的关系：
```
Δk = 2π × Δ(1/λ) ≈ 2π × Δλ/λ²
```

在波数域中，直接使用波数差：
```
d = 1 / (2n₂cos(θₜ) × Δv)
```

其中 Δv 是相邻干涉峰的波数差（cm⁻¹）。

## 三、Sellmeier色散方程

碳化硅的折射率随波长变化，使用Sellmeier方程描述：
```
n²(λ) = 1 + B₁λ²/(λ²-C₁) + B₂λ²/(λ²-C₂) + B₃λ²/(λ²-C₃)
```

对于6H-SiC，常用系数：
```
B₁ = 6.6406, C₁ = 0.0174
B₂ = 0.4530, C₂ = 1.2480
B₃ = 2.9161, C₃ = 279.920
```

## 四、最终数学模型

### 核心公式
```
厚度计算：d = 1 / (2n₂(λ)cos(θₜ) × Δv)

折射率：n₂(λ) = √[1 + Σ(Bᵢλ²/(λ²-Cᵢ))]

折射角：θₜ = arcsin(n₁sin(θᵢ)/n₂(λ))
```

### 模型特点
1. **考虑色散效应**：折射率随波长变化
2. **角度修正**：考虑入射角对折射角的影响
3. **双光束干涉**：仅考虑一次反射透射的简化模型
4. **波数域计算**：直接使用波数差，避免波长转换误差

## 五、参数范围和约束

### 物理约束
- n₁ = 1.0（空气）
- 2.5 < n₂ < 3.0（碳化硅外延层）
- 2.6 < n₃ < 2.8（碳化硅衬底）
- 0° < θᵢ < 90°（实际入射角范围）
- d > 0（厚度必须为正）

### 数值稳定性
- cos(θₜ) > 0（避免全反射）
- Δv > 0（波数差为正）
- n₂(λ) > 1（折射率物理约束）

## 六、模型验证方法

### 理论验证
1. **极限情况检验**：当d→0时，R→R₁
2. **对称性验证**：入射角变化时的物理合理性
3. **量纲分析**：确保公式量纲正确

### 数值验证
1. **蒙特卡罗模拟**：生成理论光谱验证反演精度
2. **敏感性分析**：参数变化对厚度计算的影响
3. **收敛性检验**：优化算法的收敛性

这个数学模型为后续的厚度计算算法提供了坚实的理论基础。
