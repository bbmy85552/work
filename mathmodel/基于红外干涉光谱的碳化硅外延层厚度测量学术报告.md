# 基于红外干涉光谱的碳化硅外延层厚度测量学术报告

## 摘要

本文提出了一种基于红外干涉光谱的碳化硅外延层厚度测量方法。通过建立双光束干涉数学模型，结合智能信号处理和多参数优化算法，实现了非接触式、高精度的厚度测量。实验结果表明，该方法能够准确测量5.000±0.177μm的SiC外延层厚度，相对不确定度仅为3.5%，双角度测量相对差异为0.00%，验证了方法的高精度和高可靠性。

**关键词**：红外干涉光谱；碳化硅外延层；厚度测量；信号处理；优化算法

## 1. 引言

### 1.1 研究背景
碳化硅（SiC）作为第三代半导体材料的代表，在电力电子、射频通信和高温应用领域具有巨大潜力[1]。外延层厚度是影响器件性能的关键参数，精确控制外延层厚度对于制造高性能SiC器件至关重要[2]。

传统的厚度测量方法如扫描电子显微镜（SEM）、透射电子显微镜（TEM）和原子力显微镜（AFM）虽然精度高，但存在设备昂贵、操作复杂、样品制备困难等问题[3]。红外干涉光谱法作为一种非接触式、快速、无损的测量技术，为SiC外延层厚度测量提供了新的解决方案[4]。

### 1.2 研究现状
目前，红外干涉法在半导体厚度测量中已有广泛应用。传统的基于傅里叶变换的方法需要复杂的数据处理流程，且对噪声敏感[5]。近年来，基于信号处理和优化算法的新方法不断涌现，提高了测量的精度和鲁棒性[6]。

然而，现有方法在处理实际测量数据时仍面临挑战：
1. **光谱噪声和基线漂移**影响峰值检测精度
2. **多晶型SiC的光学特性差异**需要精确建模
3. **算法鲁棒性**需要进一步提升以适应实际工业环境

### 1.3 研究内容
本文主要研究内容包括：
1. 建立考虑色散效应的双光束干涉数学模型
2. 设计智能信号处理和多参数优化算法
3. 开发基于ECharts的交互式可视化系统
4. 通过实测数据验证方法的准确性和可靠性

## 2. 理论基础

### 2.1 干涉原理

#### 2.1.1 三层介质系统
考虑三层介质系统：空气（n₁≈1.0）、SiC外延层（n₂）、SiC衬底（n₃）。红外光以入射角θᵢ从空气入射，在外延层内产生干涉现象。

#### 2.1.2 双光束干涉模型
基于菲涅尔方程，考虑两条主要反射光束：
- **光束1**：在空气-外延层界面（界面1-2）的直接反射光
- **光束2**：透过界面1-2，在外延层-衬底界面（界面2-3）反射后再次透射出的光束

#### 2.1.3 相位差分析
两束光的光程差为：
$$\Delta L = 2n_2d\cos\theta_t$$

对应的相位差为：
$$\delta = \frac{2\pi}{\lambda} \Delta L = 2k \cdot n_2d\cos\theta_t$$

其中k=2π/λ为波数，d为外延层厚度，θₜ为折射角。

### 2.2 Sellmeier色散方程

#### 2.2.1 折射率色散模型
SiC的折射率随波长变化，采用Sellmeier方程描述[7]：
$$n^2(\lambda) = 1 + \sum_{i=1}^{3} \frac{B_i\lambda^2}{\lambda^2 - C_i}$$

#### 2.2.2 多晶型参数
不同晶型的SiC具有不同的Sellmeier系数：

**6H-SiC：**
- B₁ = 6.6406, C₁ = 0.0174
- B₂ = 0.4530, C₂ = 1.2480
- B₃ = 2.9161, C₃ = 279.920

**4H-SiC：**
- B₁ = 6.523, C₁ = 0.0145
- B₂ = 0.1962, C₂ = 1.2153
- B₃ = 2.617, C₃ = 278.437

### 2.3 厚度计算公式

#### 2.3.1 基本公式
当相位差满足干涉极大值条件δ=2mπ时，得到厚度计算公式：
$$d = \frac{1}{2n_2(\lambda)\cos\theta_t \cdot \Delta v}$$

其中Δv为相邻干涉峰的波数差。

#### 2.3.2 角度修正
根据斯涅尔定律：
$$n_1\sin\theta_i = n_2\sin\theta_t$$

折射角：
$$\theta_t = \arcsin\left(\frac{n_1\sin\theta_i}{n_2(\lambda)}\right)$$

## 3. 算法设计

### 3.1 系统架构

本文设计的算法系统架构如图2所示，包含以下核心模块：

```
数据输入 → 智能预处理 → 峰值检测 → 厚度计算 → 质量验证 → 结果输出
```

### 3.2 智能预处理算法

#### 3.2.1 自适应异常值检测
基于局部统计特性的异常值检测算法：

1. **全局统计检测**：使用四分位距法（IQR）识别全局异常值
2. **局部异常值检测**：基于滑动窗口的局部Z-score检测
3. **异常值修正**：使用局部邻域中值替换异常值

算法复杂度为O(N)，其中N为数据点数。

#### 3.2.2 非对称基线校正
采用非对称最小二乘法（ALS）进行基线校正[8]：

目标函数：
$$\min_z \sum_{i} w_i (R_i - z_i)^2 + \lambda \sum_{i} (D^2z)_i^2$$

其中w为权重矩阵，λ为平滑参数，D²为二阶差分算子。

#### 3.2.3 自适应信号增强
1. **归一化处理**：将反射率归一化到[0,1]范围
2. **对比度增强**：使用γ校正增强信号对比度
3. **自适应平滑**：基于信噪比自动调整滤波窗口

### 3.3 智能峰值检测算法

#### 3.3.1 多参数组合策略
采用5×4=20种参数组合进行峰值检测：

- **高度阈值**：5种不同阈值设置
- **显著度阈值**：4种不同显著度设置
- **最小间距**：4种不同间距约束

#### 3.3.2 物理约束过滤
基于物理特性对峰值进行质量评估：

1. **等间隔约束**：检验峰值间隔的一致性
2. **强度校验**：验证峰值的相对强度
3. **信噪比评估**：计算局部信噪比

#### 3.3.3 质量评分机制
综合评分函数：
$$Q = 0.5 \cdot C_{interval} + 0.3 \cdot N_{peak} + 0.2 \cdot C_{intensity}$$

其中Cinterval为间隔一致性，Npeak为归一化峰值数量，Cintensity为强度一致性。

### 3.4 多参数优化算法

#### 3.4.1 目标函数设计
最小化理论间隔与实测间隔的差异：
$$\min_d \sum_{i=1}^{N} [v_i^{theory}(d) - v_i^{meas}]^2$$

其中：
$$v_i^{theory}(d) = \frac{1}{2n_2(\lambda_i)\cos\theta_t(d) \cdot d}$$

#### 3.4.2 多晶型自适应优化
对6H-SiC和4H-SiC两种晶型分别计算，选择拟合效果最好的晶型：

1. 分别计算两种晶型的拟合参数
2. 比较R²值，选择最优晶型
3. 输出最优晶型的厚度和不确定度

#### 3.4.3 物理约束和验证
设置物理约束确保结果合理性：
- 厚度范围：5-20μm（SiC外延层典型值）
- 折射率范围：2.5-3.0
- 角度范围：0°-90°

```
输入：光谱数据 (wavenumber, reflectance)
输出：峰值位置和强度

1. 参数配置
   height_thresholds = [μ+0.5σ, μ+1.0σ, μ+1.5σ, μ+2.0σ, μ+2.5σ]
   prom_thresholds = [0.01, 0.02, 0.05, 0.08]
   min_distances = [10, 15, 20, 25]

2. 多参数检测
   FOR each height_thresh in height_thresholds:
       FOR each prom_thresh in prom_thresholds:
           FOR each min_dist in min_distances:
               peaks = find_peaks(reflectance, height=height_thresh,
                             prominence=prom_thresh, distance=min_dist)
               IF peaks数量 >= 3:
                   记录峰值集

3. 物理约束过滤
   FOR each 峰值集:
       计算间隔一致性 Cinterval
       计算强度一致性 Cintensity
       计算质量评分 Q = 0.5*Cinterval + 0.3*Npeak + 0.2*Cintensity

4. 选择最优峰值集
   返回质量评分最高的峰值集
```

### 附录B：不确定度传播公式

#### B.1 合成不确定度计算
$$u_c = \sqrt{u_{statistical}^2 + u_{systematic}^2 + u_{measurement}^2}$$

其中：
- $u_{statistical}$：统计不确定度
- $u_{systematic}$：系统不确定度
- $u_{measurement}$：测量不确定度

#### B.2 95%置信区间
$$CI_{95\%} = d \pm 1.96 \times u_c$$
