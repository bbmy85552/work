学智AI管理平台 - 项目预览

项目概述

这是一个基于React 18 + Ant Design 5构建的现代化管理平台，提供完整的数据管理和业务操作功能。由于环境限制，无法直接运行开发服务器，但您可以通过本页面了解项目的设计和功能。

技术栈

  • 前端框架：React 18
  • 路由：React Router 6
  • UI组件库：Ant Design 5
  • 构建工具：Vite
  • Mock数据管理：自定义集中式管理系统

功能模块

数据中心

提供平台运营的核心数据指标，包含用户增长、订单统计、收入分析等关键数据的可视化展示。

学校管理

管理合作学校信息，支持学校基本信息的增删改查，包含学校规模、合作状态等管理功能。

订单中心

处理平台所有订单数据，提供订单创建、查询、状态更新等功能，支持多维度的订单筛选。

物流追踪

监控和管理所有配送订单的物流状态，支持实时物流信息更新和异常处理。

供应商管理

管理平台供应商信息，包含供应商资质、合作历史、产品目录等信息的维护。

财务管理

处理平台财务相关数据，包含交易记录、收支明细、账单管理等功能。

在本地环境运行项目

要在您自己的环境中运行并预览此项目，请按照以下步骤操作：

1. 确保已安装Node.js和npm

# 检查Node.js和npm版本 node -v npm -v

2. 安装项目依赖

# 进入项目目录 cd c:\Users\Administrator\Documents\trae_projects\AI # 安装依赖 npm install

3. 启动开发服务器

# 启动开发服务器 npm run dev

4. 访问项目

打开浏览器，访问终端中显示的本地地址（通常是 http://localhost:5173/）
