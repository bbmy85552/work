### 四、预期可视化图表（建议4类）
#### 1. “实践-技术-满意度”三维交互空间分布图 (3D Response Surface / Scatter Plot)
+ **图表形式：** 三维散点图（3D Scatter）或三维曲面图（3D Surface）。
+ **坐标轴设定：**
    - X轴：**实践强度（A）**
    - Y轴：**数字技术融合度（E）**（或D）
    - Z轴（高度）：**满意度（因变量）**
+ **展示重点：**
    - **路径可视化：** 直观呈现随着X和Y的提升，Z轴（满意度）是否呈现“爬坡”趋势，验证正向影响路径。
    - **极端区识别：** 标记出“低实践-低技术-低满意度”的**塌陷区**，以及“高实践-高技术-高满意度”的**峰值区**。
    - **交互效应：** 观察曲面是否存在扭曲或陡峭变化，以此判断“实践”与“数字技术”是否存在显著的交互增强效应（例如：在高数字技术下，实践对满意度的提升更明显）。

#### 2. 变量共现与耦合机制网络拓扑图 (ERGM Network Topology Graph)
+ **图表形式：** 网络关系图（节点-边模型），基于指数随机图模型（ERGM）生成。
+ **图示元素：**
    - **节点（Nodes）：** 代表各个变量（满意度、实践强度、数字技术、控制变量等）。节点大小代表其在网络中的**中心度（Centrality）**，即影响力大小。
    - **连线（Edges）：** 代表变量间显著的条件依赖或共现关系。连线粗细代表关联强度（概率比）。
+ **展示重点：**
    - **耦合机制：** 揭示“实践强度”与“数字技术”是否作为**核心节点（Hubs）**紧密连接了“满意度”，形成稳定的三角形闭合结构。
    - **桥接作用：** 展示数字技术变量是否作为“桥梁”，连接了其他控制变量与满意度，证明其调节或中介作用。
    - **社群结构：** 识别变量间是否形成了特定的功能模块（如“技术-实践”模块），体现系统的模块化特征。

#### 3. 多模型特征重要性排序对比图 (Multi-Model Feature Importance Bar Chart)
+ **图表形式：** 堆叠条形图或并列条形图。
+ **图示内容：**
    - 横轴：特征重要性分数（Importance Score / Coefficient）。
    - 纵轴：各自变量名称（按影响力从高到低排列）。
    - 图例：随机森林、XGBoost、Lasso回归（不同颜色代表不同模型）。
+ **展示重点：**
    - **一致性验证：** 直观展示在不同算法下，**A（实践）**、**D/E（数字技术）**是否始终占据Top 3的位置，证明关键因子的鲁棒性。
    - **解释力差异：** 对比线性模型（Lasso）与非线性模型（RF/XGBoost）的排序差异，挖掘变量间可能存在的非线性贡献。

#### 4. 实际值与预测值拟合效果散点图 (Actual vs. Predicted Satisfaction Scatter Plot)
+ **图表形式：** 散点回归图。
+ **图示内容：**
    - X轴：实际调研的满意度分数。
    - Y轴：机器学习模型预测的满意度分数。
    - 辅助线：45度对角线（y=x，代表完美预测线）。
+ **展示重点：**
    - **模型效度：** 散点越紧密围绕对角线分布，说明模型（随机森林/XGBoost）对满意度的解释能力越强（R²高）。
    - **偏差分析：** 通过观察离群点，分析是否存在某些特殊样本（如特定专业或背景的学生）使得模型预测失效，为研究局限性讨论提供素材。
