# 金课问卷数据分析系统技术文档

## 1. 项目概述

### 1.1 项目背景
本项目基于金课程问卷调查数据，构建了一个完整的Python数据分析系统。系统处理学生参与度、学习感知、满意度等多维度调查数据，提供数据清洗、统计分析、可视化等核心功能。

### 1.2 数据源描述
- **变量表.csv**: 包含42个题目项目，涵盖11个潜变量维度
- **数据(1).csv**: 包含579条有效调查记录，47个数据字段
- **调查规模**: 579名学生参与，涵盖性别、年级、专业等人口统计学变量

### 1.3 技术栈
- **uv**:虚拟环境
- **Python 3.12**: 核心开发语言
- **Pandas**: 数据处理和分析
- **NumPy**: 数值计算
- **Echarts**: 数据可视化
- **Scikit-learn**: 机器学习分析
- **SciPy**: 统计分析

## 4. 量表维度设计

### 4.1 潜变量结构
基于变量表分析，系统识别出以下11个潜变量：

1. **A(实践环节强度)**: 3个题目 (A-1, A-2, A-3)
2. **B(学习参与度)**: 3个题目 (B-1, B-2, B-3)
3. **C(考核合理性)**: 3个题目
4. **D(数字技术接受度)**: 3个题目
5. **E(数字技术融合度)**: 3个题目
6. **F(学习期望)**: 5个方面
7. **G(课程形象)**: 5个方面
8. **H(学习感知)**: 6个方面
9. **I(学习表现)**: 5个方面
10. **J(满意度)**: 5个方面

### 4.2 量表计分规则
- **5点李克特量表**: 1=非常不同意, 2=不同意, 3=一般, 4=同意, 5=非常同意
- **反向计分**: 对负向表述题目进行分数转换
- **维度得分**: 同维度题目得分的平均值

## 5. 核心算法实现

### 5.1 Cronbach's Alpha信度分析
```python
def cronbach_alpha(df_items):
    """计算Cronbach's Alpha系数"""
    k = len(df_items.columns)
    if k <= 1:
        return 0

    item_variances = df_items.var()
    total_variance = df_items.sum(axis=1).var()

    alpha = (k / (k - 1)) * (1 - item_variances.sum() / total_variance)
    return alpha
```

### 5.2 描述性统计

```python
def comprehensive_descriptive_stats(data):
    """综合描述性统计"""
    stats = {
        'count': data.count(),
        'mean': data.mean(),
        'std': data.std(),
        'min': data.min(),
        '25%': data.quantile(0.25),
        '50%': data.median(),
        '75%': data.quantile(0.75),
        'max': data.max(),
        'skewness': data.skew(),
        'kurtosis': data.kurtosis()
    }
    return pd.DataFrame(stats)
```

### 5.3 相关性分析
```python
def correlation_analysis(data, method='pearson'):
    """相关性分析"""
    corr_matrix = data.corr(method=method)

    # 计算显著性水平
    from scipy.stats import pearsonr
    p_values = pd.DataFrame(index=data.columns, columns=data.columns)

    for col1 in data.columns:
        for col2 in data.columns:
            if col1 != col2:
                _, p_val = pearsonr(data[col1], data[col2])
                p_values.loc[col1, col2] = p_val

    return corr_matrix, p_values
```

## 7. 使用示例

### 7.1 基本使用
```python
from main import QuestionnaireAnalyzer

# 初始化分析器
analyzer = QuestionnaireAnalyzer(
    data_path='数据(1).csv',
    variables_path='变量表.csv'
)

# 运行完整分析
results = analyzer.run_full_analysis(output_dir='analysis_results')

# 生成报告
report_path = analyzer.generate_report(format='html')
print(f"分析报告已生成: {report_path}")
```

### 7.2 自定义分析
```python
# 加载数据
data, variables = analyzer.loader.load_data()

# 信度分析
scale_items = ['实践环节强度1', '实践环节强度2', '实践环节强度3']
alpha = analyzer.analyzer.reliability.cronbach_alpha(scale_items)
print(f"Cronbach's Alpha: {alpha:.3f}")

# 分组比较
group_stats = analyzer.analyzer.statistics.group_comparison('性别')

# 可视化
analyzer.visualizer.plot_distribution(data['满意度—教学方式'])
```
